Алгоритм написан на python 3.7 для анализа текстов произведений.

Перед запуском алгоритма необходимо записать в файл in.txt текст, который следует обрабатывать. В папке находятся примеры текстов для проверки работы программы. "Theatre" - 
произведение "Театр" Соммерсета Моэма на английском языке, "RedHat" - Красная шапочка, "WarPiece" - Война и мир Л.Н. Толстого.

Данный алгоритм принимает на вход текстовый файл in.txt, обрабатывает его и выводит массив всех слов из словаря, общее количество слов, а также сам словарь с 
количеством использований слова в тексте, частотностью использований слова и процентным содержанием к общему количеству слов.

После работы программы, алгоритм сам откроет файл с результатами работы.

----------------------------------------------------------------------------------------------

The algorithm is written using Python 3.7 to analyze the texts.

Before running the algorithm, it is necessary to write the text to be processed into the in.txt file. The folder contains examples of texts to test the program. Theatre-
work "Theater" by Somerset Maugham in English, "RedHat" - Little Red Riding Hood, "WarPiece" -
War and Peace by L.N. Tolstoy in Russian.

This algorithm takes a text file in.txt as input, processes it and outputs an array of all words from the dictionary, the total number of words, and the dictionary itself with
the number of uses of the word in the text, the frequency of use of the word and the percentage of the total number of words.

After running the program, the algorithm will open the file with the results of the work by itself.
